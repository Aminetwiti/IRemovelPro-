## CONCLUSION — Comment iRemoval PRO jailbreak l'iPhone
================================================================================

### Phase 1 : Mise en DFU (Device Firmware Update)
- Détection iPhone via USB (libimobiledevice)
- Mode DFU via USB control transfer

### Phase 2 : Exploit checkm8 (A5-A11) ou palera1n (A12-A16)
- **checkm8** = exploit bootrom non-patchable (axi0mX, 2018)
- Heap buffer overflow dans le code DFU de la SecureROM
- Exécute du code en mode Pongo (avant iBoot)
- Affecte A5-A11 (iPhone 4S à iPhone X)
- **palera1n** = fork checkra1n pour A12+

### Phase 3 : Injection du kernel jailbreaké
- Kernel custom poussé via USB
- Désactive AMFI, code signing, trust cache, sandbox

### Phase 4 : SSH + installation Cydia
- OpenSSH sur port 22 (root:alpine)
- Cydia + MobileSubstrate installés via dpkg/apt

### Phase 5 : Déploiement blackhound.dylib
- Tweak `com.panyolsoft.blackhound` poussé via SCP
- Installé dans `/Library/MobileSubstrate/DynamicLibraries/`
- 5 hooks s'activent au boot

### Phase 6 : Bypass Activation Lock
- PC contacte s13.iremovalpro.com (iact8.php)
- Serveur signe faux ticket avec RSA-1024
- Ticket poussé via SSH vers /var/mobileactivationd/
- Hooks acceptent le ticket forgé
- iPhone démarre sans iCloud lock
